snowlloons
==========
Balloons Live! Wallpaper
Relaxing balloons drifting into the air on sunny days.

Features
--------
Pop the balloons!
Customize the balloon colors by picking from 9 colors or random!

Requires Android 2.1 and devices that support Live! Wallpaper.
